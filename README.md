# edge-tts-gui
一个简单的edge-tts的GUI，无需安装任何依赖项，只需下载可执行文件即可。

新版本支持GPT-SoVITS api_v2

快捷键：
- `Ctrl` + `Enter`：播放
- `Ctrl` + `S`：保存

全局快捷键：
- `F9`：快速朗读选中文本（或已截图片）

便捷操作：
- 选中文本后，在选中文本末尾会出现悬浮播放按钮，点击即可朗读（同 `F9` 效果）；朗读中按钮会变为停止，点击可停止
